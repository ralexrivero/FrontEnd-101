Code snippet

Escapando caracteres especiales, utilizando elementos pre y code, junto con CSS formateamos nuestro codigo para generar un code snippet dark mode, estilo VS Code

              <meta charset="UTF-8">
              <meta http-equiv="X-UA-Compatible" content="IE=edge">

              <link rel="stylesheet" href="style.css">

            </head>
